# Electronics Store Management System

مشروع مبني باستخدام لغة C# يعتمد على مفاهيم البرمجة الكائنية (OOP) لمحاكاة نظام إدارة متجر إلكترونيات من خلال Console Application.

يهدف المشروع إلى تنظيم عمليات إدارة الأجهزة والموظفين داخل المتجر مع تطبيق نظام صلاحيات يفرق بين المدير والموظف.

---

## Features

### صلاحيات المدير (Manager)
- عرض جميع الأجهزة
- البحث عن جهاز
- إضافة أجهزة جديدة
- حذف جهاز موجود
- إضافة موظفين
- حذف موظفين
- عرض جميع الموظفين

### صلاحيات الموظف (Employee)
- عرض الأجهزة المتوفرة
- البحث عن جهاز
- استخدام النظام بصلاحيات محدودة
